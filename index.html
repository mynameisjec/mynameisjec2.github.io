Staff Commands

/announce [text]

Aliases: /announcement

Description: Sends the specified text (can be colored) to all players in the game.

ㅤ

/clearchat

Aliases: /cc

Description: Clears everybody's chat by sending a lot of empty messages.

ㅤ

/mutechat

Aliases: /stopchat, /togglechat

Description: Mutes the chat; normal players will be unable to send messages.

ㅤ

/staffchat [text]

Aliases: /sc

Description: Sends the specified message in the staff chat; only staff with permission can see it.

ㅤ

/clearchat

Aliases: /cc

Description: Clears everybody's chat by sending a lot of empty messages.

/reportlist

Aliases: None

Description: Returns the latest 10 (can be changed) reports; not for specific player, but all.

ㅤ

/getreports [player]

Aliases: None

Description: Returns the latest 10 (can be changed) reports about the specified player.

ㅤ

/clearreports [player]

Aliases: None

Description: Clears the specified player's reports (all of them); removed from /getreports and /reportlist.

ㅤ

/reportreward [player]

Aliases: None

Description: Gives the specified player the report reward (changeable, and disable-able). If they are not online, they will receive it on join.

/settime [player] [seconds]

Aliases: None

Description: Sets the specified player's time to the specified amount.

/suspect [player]

Aliases: None

Description: Teleports player to jail, and tells them to join General VC 3 or they will get banned. This is used if a staff member is unsure if somebody is hacking or not.

ㅤ

/unsuspect [player]

Aliases: None

Description: If the specified player is in the jail (from /suspect), they will be taken out and will be thanked for cooperating.

ㅤ

/shadowban [ban/remove] [player]

Aliases: None

Description: Shadow bans the player; makes it so they can talk, but no messages will be seen by anybody except them.

ㅤ

/sbansee [on/off]

Aliases: None

Description: When enabled, if a shadow banned player speaks, the user will be notified (messages will not be shown, only notification in chat).
